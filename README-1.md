# Školské obvody MŠ — Zobor, Nitra

Interaktívna mapa rozdelenia ulíc Zobora do 4 obvodov materských škôl.

## Obvody

- 🔴 **MŠ Dobšinského 2885/8** (53 ulíc) — centrum Zobora
- 🟢 **MŠ Vansovej 6** (28 ulíc) — severovýchod
- 🟠 **MŠ Platanová 3** (38 ulíc) — severozápad / Dražovce
- 🟣 **MŠ Belopotockého 2** (38 ulíc) — juhovýchod
